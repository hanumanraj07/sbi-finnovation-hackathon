 🛡️FraudShield – Pre-Click Fraud Detection System
🧠 Overview

FraudShield is an AI-powered mobile security application designed to protect users from phishing attacks and fake banking applications. It detects malicious links and fraudulent APK files before users open or install them, preventing financial fraud and data theft.

With the rise of fake SBI YONO apps and phishing scams via SMS/WhatsApp, this system acts as a real-time digital safety layer for users.

🎯 Problem Statement

Cybercriminals are distributing fake banking apps and phishing links that mimic trusted platforms like SBI YONO. These attacks trick users into sharing sensitive information such as OTPs, MPINs, and login credentials, leading to financial loss and identity theft.

💡 Solution

FraudShield provides a prevention-first approach by scanning links and APK files before execution. It alerts users instantly with a Trust Score and explains the risk, helping them make safe decisions.

✨ Key Features

🔗 Link Scanner
Detects phishing and malicious URLs
Checks domain authenticity
Identifies suspicious keywords

📦 APK Scanner
Scans APK files before installation
Verifies app signature and package name
Detects fake banking apps

🧠 Trust Score System
Provides a score (0–100) for safety
Color-coded risk levels:
🟢 Safe
🟡 Suspicious
🔴 Dangerous

🚨 Real-Time Warning System
Alerts users before opening risky links
Provides reasons for detection

📢 Explainable Detection
Shows why a link/app is dangerous
Builds user awareness and trust
